## Présentation

Site vitrine one-page pour **Maison Claire Débarras**, entreprise de débarras basée à L'Haÿ-les-Roses (Val-de-Marne, 94). Intervention rapide et éco-responsable sur toute l'Île-de-France.

## Stack technique

| | |
|---|---|
| **Site** | HTML / CSS / JS — une seule page, zéro framework |
| **Hébergement** | GitHub Pages |
| **Domaine** | OVH |
| **Formulaire** | EmailJS (envoi de devis + confirmation client) |
| **SEO** | Meta tags, Open Graph, JSON-LD, sitemap, robots.txt |

## Structure du projet

```
├── index.html            # Page principale
├── CNAME                 # Domaine personnalisé
├── robots.txt            # Directives moteurs de recherche
├── sitemap.xml           # Plan du site
├── favicon.ico           # Favicon
├── apple-touch-icon.png  # Icône iOS
└── images/
    ├── logo.jpg          # Logo principal
    ├── logo-sans-fond.png
    ├── icon-192.png      # PWA icons
    ├── icon-512.png
    ├── camion.jpeg
    └── photos/           # Photos du site (Unsplash)
```

## Déploiement

Le site se déploie automatiquement via **GitHub Pages** à chaque push sur `master`.
